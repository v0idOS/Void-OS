Install and setup guide

End-to-end setup flow for Void OS playbook deployments.

Requirements

System

  • Supported Windows build
  • Administrator access
  • Reliable internet connection
  • Sufficient free disk space

Recommended prep

  • Create a restore image / backup
  • Test first in a virtual machine
  • Disconnect non-essential devices during install
  • Keep a fallback installer/recovery media ready

Installation steps

1

Download latest .apbx from the releases page.

2

Launch AME Wizard and load the playbook file.

3

Select install options (Defender, mitigations, browser).

4

Start deployment and wait for completion/reboot prompts.

Verification checklist

Release file verification

  • Compare SHA/hash from release notes
  • Confirm file size matches published asset
  • Download only from official repository links

Post-install checks

  • Open key desktop shortcuts and settings pages
  • Confirm selected browser installation (if chosen)
  • Run basic network/audio/GPU sanity checks

Troubleshooting

Wizard fails before deployment starts

Re-download the artifact, verify checksum, and test with antivirus exclusions or in a VM.

A selected option did not apply

Re-run the relevant script/toggle from desktop utilities, then reboot and retest.

Need help with a specific machine

Open an issue with hardware specs, Windows build, and install logs for faster triage.
