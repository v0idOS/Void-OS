FAQ

Is Void OS a separate operating system?

No. It is a Windows playbook/configuration project that applies a curated set of tweaks and scripts.

Can I audit what gets changed?

Yes. The project is source-available in GitHub and tweak logic is defined in YAML/PowerShell/cmd files.

Can I fork and customize it?

Yes. Fork the repository, change modules or tweaks, then build your own playbook artifact.
