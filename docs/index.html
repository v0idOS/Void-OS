What It Is

Void OS is not a custom ISO. It is an aggressive, open-source AME Wizard playbook that systematically dismantles Windows bloat and injects a custom Master Engine to manage core system state. Instead of relying on a black-box modified image, Void natively unbinds unnecessary services, secures telemetry leaks, and drops a complete rollback toolkit directly onto your desktop.

Features

01

Dynamic Core Scheduling

Dynamically rewrites Win32PrioritySeparation to prioritize absolute foreground execution, with automated compensation for Intel Hybrid architectures to prevent Thread Director stutters.

02

Aggressive Debloat

Automated removal of diagnostic tracking, telemetry subsystems, Copilot, web search integration, and forced updates via a native deployment mechanism.

03

Native Standby Cleaner

An embedded memory module that asserts system privilege to automatically flush standby lists without requiring bloated third-party tools like ISLC.

04

Power User Toggles

A granular toolkit injected into your C:\ drive, allowing you to instantly revert security configurations, reinstall missing applications, or reverse registry tuning.
